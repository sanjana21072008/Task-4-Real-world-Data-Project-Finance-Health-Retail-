# Real-world Data Project (Retail Sales Analysis)

## Objective
To analyze retail sales data and identify best performing products using basic data analysis techniques.

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
A sample retail dataset containing product names, price, and quantity.

## Steps Performed
1. Created a sample retail dataset.
2. Calculated total sales for each product.
3. Used groupby to find total sales by product.
4. Visualized product performance using a bar chart.

## Findings
- Total sales were calculated for all products.
- Best selling products were identified.
- Pen and Book showed higher sales compared to others.
- Visualization helped understand sales distribution clearly.

## Outcome
Gained experience in real-world data analysis and visualization using Python.
